Athame

Athame is a program for downloading music from streaming services like Tidal and Google Play Music. It is open-source and free for personal use. It is currently in alpha and requires Microsoft Windows and .NET (though it does kind of work on Linux with Mono).

Disclaimer & download

  • Athame is alpha software, which means it could break at any time.
  • You must not share or redistribute any content downloaded with Athame.
  • Athame attempts to replicate the function of the "offline download" feature many music streaming apps have. Therefore, when your music streaming service subscription ends, you are obliged to delete any content you have downloaded to your computer or device.
  • Athame uses internal APIs, which means they are subject to change at the discretion of the streaming service provider.

  • Athame 0.4.6

    0.4.6

    The plugin API has not changed, so plugins that worked in 0.4.5 will work in this version.

    New users are advised to download the zip file ending in -base-plugins.zip. Otherwise, the zip file ending in the version number can be unzipped directly into your existing program directory.

    • #74 et al - Fix messed up default playlist path format. Existing users need to change the path format to {PlaylistName}/{Title} - {AlbumArtistOrArtist} or a similar format for this to work.
    • #85 - long file names. See this article for enabling long file names on Windows 10 Anniversary Edition or later. Note that the code that tries to truncate long file names on earlier platforms is primitive, and may not work if you have a long download destination.
    • #93 - groups now display the total number of tracks in them, as well as the total number of available tracks.
    • #96 - there is now an option in settings to download a M3U or PLS playlist of each collection queued. Due to the way the path format system works, the playlist file is saved in the destination directory. For instance, if the destination directory is your Music folder and your path format is {Artist}/{Album}, the playlist will be saved in your Music folder, and named the same as the collection.
    • #97 - Athame now alerts you if there are items in the queue that haven't been downloaded yet. There is an option to disable this in settings.
    • #95 - The queue can now be scrolled while downloading.
    • #99 - Fixed the issue where the optional "Genre" track property wasn't checked for null before being set in the tag.
    • Fixed issue where the album artwork download option {Track} {Album}.ext doesn't validate the album artwork path.
    • Fixed issue where exception dialog doesn't appear on double click
    • Added an option for album art to not be saved when downloading playlist
    • Update licenses in about box

    A sincere thank you to everyone who starred this repo, filed an issue, downloaded a copy, or gave a donation/thank you. Your continued support motivates me to continually improve Athame in my limited spare time, and without it I would've given up months ago.

  • Athame 0.4.4 / Plugin API 2

    0.4.4

    You will need to delete your Tidal config file (%AppData%\Athame\Plugin Data\Tidal Settings.json), since the new Tidal plugin uses a different API key to enable master quality downloads.

    • Fix bug where downloading an album with no artwork would cause an error.
    • Fix bug where the Google Play Music plugin wouldn't work due to being unable to load the JSON library.
    • Better plugin load error handling.
    • Files are now downloaded with prefix -temp on the filename, then renamed to the correct name when completed. This is to prevent programs like the Google Play Music Manager from attempting to upload a file while it's still being downloaded.
    • Tidal plugin: Enable master quality (MQA) downloads. Just choose the "Master" quality option in settings.
    • You will need to delete your Tidal config file (%AppData%\Athame\Plugin Data\Tidal Settings.json), since the new Tidal plugin uses a different API key to enable master quality downloads.
    • Developers: Plugin API version 2 is now supported, which introduces the Picture abstract class instead of the CoverUri property, as well as paginated records. See the documentation wiki for more details.

  • Athame 0.4.3 / Plugin API 1

    0.4.3-1

    Note: This release will reset your settings.

    • Fix issue where loading plugins on a system with .NET 4.7 installed would cause a NotSupportedException to be thrown.
    • Remove type information from main settings, and put plugin settings in a separate file. This fixes the issue where settings cannot be parsed after a plugin is removed.
    • Breaking changes for the plugin API.

    The default release archive does not contain any plugins by default. For legacy users, the athame-0.4.3-base-plugins.zip release is available, which includes the Tidal and Google Play Music services.

    Both the Tidal and Google Play Music service plugins are available from their respective repositories, on the Releases tab. In the future, I will also promote other plugins that can be optionally installed.

    To install a plugin, just extract it into the "Plugins" folder in the same directory as the executable.
